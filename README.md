# jaeminlang

재민랭은 한국어로 코딩할 수 있는 프로그래밍 언어입니다.  

|대표적인 키워드| 
|-----|
|안산|
|재민|
|그램|
|러스트|

## 실행

재민랭 소스코드를 실행하시려면 릴리즈로 가셔서  
자신의 OS에 맞게 실행 파일을 다운로드 하신 후  

리눅스의 경우:
```sh
chmod +x jaeminlang # 권한 주기 (한번만 실행)

./jaeminlang source.jml
```

윈도우의 경우:  
```cmd
jaeminlang.exe source.jml
```
